# 99-commands

리눅스 계열에서 자주 사용하는 명령어 목록

- man (또는 {command} --help)
- less
- more
- grep
- egrep [*]
- df
- du
- ncdu [*]
- kill
- ps
- curl [*]
- wget [*]
- chmod
- chown
- usermod
- adduser / useradd
- sed
- vi / vim
- {command}& (백그라운드에서 명령어 실행)
- screen [*]
- crontab
- echo "" > /dev/null (`stdout` 버리기)
- systemctl / service
- uname -a
- cat /etc/os-release


---
[*] 추가 패키지 설치가 필요할 수 있음
